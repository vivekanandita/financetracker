Connect your GitHub repo

This ledger stores its data as a JSON file directly in a GitHub repo you own, via the GitHub API. Nothing is sent anywhere else.

Use a fine-grained token scoped only to this one repo, with "Contents: Read and write" permission. Generate one at github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens.

The token is stored only in this browser's local storage — never written into any file you commit.
